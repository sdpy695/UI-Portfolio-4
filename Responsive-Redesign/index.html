Responsive Redesign:

The Belgian-American Embassy

One of the best current resources for Beglian-American expats is the website for the Belgian Embassy and Consulates in the United States. However, the information on the current website is disorganized and particularly difficult to view on a mobile device. I reimagined the page to be more accessible.

The Original Site

Accessibility

Several fundamental issues with this site make it inaccessible to the target users:

  • The page is full of small text with low contrast against its background
  • The images included don't have alt text, so they can't be interpreted by a screenreader
  • On the mobile version of the site, the buttons are too small for even the most dextrous fingers to reach them on a touchscreen.
  • The language bar gets lost in the corner, allowing for users to be trapped in a version of the site that's in a language they don't understand. In combination with the small buttons, users with less mobility aren't able to access the language change feature.

    The final design is a dummy page made using html and css which is displayed in a more accessible way and can responsively adapt to screen resizing.

    In this project, I learned a lot of nitty gritty details about resizing a page using just HTML and css. I found it really difficult to design something beautiful for a government agency. It felt that every design choice I made would be a public statement. Knowing this design is ugly and not as function as I'd like, I've started a redesign of the redesign.
